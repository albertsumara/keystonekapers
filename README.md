# KeystonKapers Clone

**KeystonKapers Clone** is a platformer game built in Pygame, heavily inspired by the classic *Keystone Kapers*.  
In this game, the player chases an AI opponent across three platforms, avoiding obstacles and collecting bonus points.

## Gameplay

- Move across **three platforms** using **moving stairs** and **elevators**.  
- Encounter **various obstacles** moving at different heights that make chasing the opponent more challenging.  
- Collect **bonus points** in the form of **chests** or **banknotes**.  
- The round ends when you **catch the opponent (bot)**.  
- Each subsequent level becomes more difficult and fast-paced.

## Controls

- **A →** Move left  
- **D →** Move right   
- **W →** Jump / Enter stairs / Enter elevator  
- **S →** Crouch / Exit elevator
